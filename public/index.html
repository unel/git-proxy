Welcome to the Git Proxy Service - a Cloudflare Worker that provides API access to GitHub repositories.

Available Endpoints

GET /files/o/:owner/r/:repo/:path

Specify both owner and repository.

GET /files/r/:repo/:path

Use default owner: {{DEFAULT_OWNER}}

GET /files/:path

Use defaults: {{DEFAULT_OWNER}}/{{DEFAULT_REPO}} (branch: {{DEFAULT_BRANCH}})

Usage Examples:

/files/o/facebook/r/react/README.md - Explicit owner and repo

/files/r/modular/README.md - Default owner ({{DEFAULT_OWNER}})

/files/worker.js - Defaults ({{DEFAULT_OWNER}}/{{DEFAULT_REPO}})
